# 🐾 Mario Adventure Game

A fun Mario-style platformer game built with HTML5 Canvas and JavaScript!

## 🎮 How to Play

1. **Open `index.html`** in your web browser
2. **Controls:**
   - **Arrow Keys**: Move left and right
   - **Spacebar**: Jump
3. **Objective:** Collect all the golden coins while avoiding the red enemies!

## 🎯 Game Features

### Player (Red Mario)
- Smooth movement with arrow keys
- Jumping mechanics with gravity
- Collision detection with platforms
- Visual direction indicators (eyes and hat)

### Platforms
- Multiple platforms to jump on
- Solid collision detection
- Brown wooden appearance

### Collectibles
- **Golden Coins**: Collect them to increase your score
- Each coin gives you 10 points
- Visual shine effect on coins

### Enemies
- **Red Enemies**: Avoid them or lose points
- Moving enemies that bounce off screen edges
- Collision with enemies knocks you back and reduces score

### Scoring System
- **Coins**: +10 points each
- **Enemy collision**: -5 points
- **Falling off screen**: -10 points and respawn

### Visual Elements
- Beautiful gradient background (sky to grass)
- Decorative clouds
- Smooth animations
- Mario-style character design

## 🚀 Getting Started

1. Download all files (`index.html`, `game.js`, `README.md`)
2. Open `index.html` in any modern web browser
3. Start playing immediately!

## 🛠️ Technical Details

- **Built with:** HTML5 Canvas, Vanilla JavaScript
- **No external dependencies** - runs in any modern browser
- **Responsive design** with beautiful styling
- **Smooth 60fps gameplay** using requestAnimationFrame

## 🎨 Customization

You can easily modify the game by editing `game.js`:
- Change player speed, jump power, or appearance
- Add more platforms, coins, or enemies
- Modify scoring system
- Adjust colors and visual effects

## 🎉 Have Fun!

Jump around, collect coins, and try to get the highest score! The game features classic Mario-style platformer mechanics with smooth controls and engaging gameplay.
